# Social Auth Package

A Flutter package for social authentication with Google and Apple.

## Features
✅ Google Sign-In  
✅ Apple Sign-In  
✅ Unified API Interface  

## Installation
Add this to your `pubspec.yaml`:
```yaml
dependencies:
  social_authentication:
    git:
      url: https://github.com/AhmedHusseinHafez/social_auth.git
      ref: main
